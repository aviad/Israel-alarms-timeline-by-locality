* Architecture

** =alarms_core.py= — shared pure-Python logic (~563 lines, was 2184)
- =render_chart(times, label, bin_hours, start_date, data_cutoff, style, fmt, forecast, all_records, city_filter, ...)= → SVG bytes
  - Pure SVG string building — no matplotlib/numpy
  - Tufte-style: EB Garamond font, night shading, lines/dots modes
  - Forecast annotation: economist-style curved line + label
  - Night mode (8pm–6am): switches label to "tonight (until 7am)"
- =compute_prediction(...)= — extracted for header/desc embedding
- Re-exports everything from split modules for backward compatibility
- No external dependencies — pure stdlib

** =city_translations.py= — city name mapping
- =CITY_TRANSLATIONS= — 1627 Hebrew→English city names

** =israel_time.py= — Israel time utilities
- =_israel_utc_offset()=, =_epoch_to_israel()= — Israel DST (last Friday of March rule)

** =data_loading.py= — CSV and API loading
- =load_alerts(csv_text, area_filter, threat, start)= → =(times, seen_ids)=
- =load_api_alerts(api_data, area_filter, threat, start, seen_ids)= → =times=
- =load_alerts_rich= / =load_api_alerts_rich= — event-level records for Ridge model
- =ROCKET_DESC= constant

** =forecast.py= — prediction models (pure Python)
- =predict_remaining(times, now, method)= — "simple" (3-feature linear) or "advanced" (rate-sub blend)
  - Rate-sub: predict daily total, subtract observed
  - Blend: w×rate_sub + (1-w)×rate, w=hours_remaining/24
  - Backtest MAE: blend 1.54, rate-sub 1.81, rate 1.63
- =predict_remaining_ridge(all_records, city, now, alpha=10)= — 34-feature Ridge regression
  - Rate-sub framing, 8 cutoff hours × prior days training
  - Features: wave stats, interaction terms, global/city-level
- =predict_night_rolling(times, now)= — 7-night rolling average (8pm–midnight fallback)
- =predict_night_ridge(all_records, city, now)= — 7am–7am Ridge (midnight–6am)
- Night mode selection: Ridge beats naive by 35% after midnight; rolling average before midnight

** =alarms_graph.py= — CLI wrapper
- argparse, requests, file-based cache
- PNG export via cairosvg with embedded ETBembo fonts as base64
- =--forecast off/simple/advanced/ridge=

** =worker/entry.py= — Cloudflare Worker
- =WorkerEntrypoint= with =async def fetch(request)=
- Landing page: Hebrew searchable combobox (1627 cities), date picker, lines/dots, threat filter (rockets/UAVs/both), forecast radio (off/simple/full)
- Two-phase loading: chart renders immediately, forecast overlaid async
- Progress messages during load
- Prediction displayed as stat card above chart + embedded in SVG =<desc id="pred-data">=
- KV caching: CSV 30-min TTL, API 2-min TTL, global features 30-min TTL
- CSV pre-filtered to rows ≥ 2026-02-27 (full 122K-row CSV exceeds 128MB)
- SVG/PNG download buttons, copy icon, PNG rasterized client-side via canvas
- Mobile-responsive: stacked form, 44px touch targets, scrollable chart, deeplinks
- Credits footer: yuval-harpaz/alarms, forecast model, Tufte style
- Deeplinks: URL params auto-populate form and generate chart

* Lessons Learned

- =WorkerEntrypoint= only (=on_fetch= deprecated since wrangler 4.71.0)
- =Response(bytes)= → garbled; must decode to =str= first
- SVG in =<img>= blocks =@import=; use =<object>= instead
- =zoneinfo= needs =tzdata= in Pyodide; moved to CLI only
- =datetime.fromtimestamp()= uses system TZ (UTC on Cloudflare); use explicit conversion
- Israel DST: last *Friday* of March (not Sunday like EU)
- Canvas sandboxes =<object>= SVGs; must fetch+embed Google Fonts as base64 for PNG export
- 8pm Ridge forecast overfits with ~14 days training data; rolling average is better fallback
